# Landing Page - NR1

Esta Landing Page foi cuidadosamente projetada para apresentar a solução **NR1**, indo além de uma simples vitrine: ela funciona como um canal estratégico para comunicar valor, gerar confiança e converter visitantes em leads qualificados.

O objetivo principal é simplificar a comunicação sobre gestão psicossocial nas empresas, demonstrando de forma clara os diferenciais da plataforma e incentivando o contato com especialistas.

Com essa presença online, a NR1 proporciona:

* Vitrine da solução no ambiente digital;
* Maior visibilidade e autoridade no mercado;
* Comunicação clara sobre a proposta e funcionalidades da plataforma;
* Geração de leads qualificados;
* Melhor posicionamento em mecanismos de busca;
* Facilidade para empresas entenderem a solução;
* Transmissão de confiança e profissionalismo desde o primeiro contato.

### 📌 Detalhamento da Pasta src/

| Diretório/Arquivo |                                                         Descrição                                                         |
| ----------------- | :-----------------------------------------------------------------------------------------------------------------------: |
| components/       | Contém todos os componentes da interface, organizados por seções da landing page, garantindo modularidade e reutilização. |
| styles/           |                            Centraliza os estilos globais e configurações de SASS e CSS Modules.                           |
| App.jsx           |                 Componente raiz da aplicação, responsável por estruturar e renderizar as seções da página.                |

### 📥 Como baixar o projeto na sua máquina

Siga o passo a passo abaixo para clonar e rodar o projeto localmente:

1. **Clone este repositório**

   ```
   git clone <URL_DO_REPOSITORIO>
   ```

2. **Acesse a pasta do projeto**

   ```
   cd nr1
   ```

3. **Instale as dependências**

   ```
   npm install 
   ```

   ou

   ```
   yarn install
   ```

4. **Execute o projeto em ambiente de desenvolvimento**

   ```
   npm start 
   ```

   ou

   ```
   yarn start
   ```

5. **Abra no navegador**

   ```
   http://localhost:3000
   ```

A porta pode variar dependendo da configuração do ambiente.

Pronto! O projeto estará rodando na sua máquina. 🚀
